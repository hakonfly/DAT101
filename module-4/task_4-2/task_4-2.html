        What is an array?
        
    An array is an ordered list of values stored in one variable. Each element has an index, so we can read or change values by their position.

        What does it mean to "hardcode" when we talk about programming?
        
    Hardcoding means writing the values directly into the source code instead of reading them from input, a file, or a database. It is useful for simple examples, tests, and fixed values.

        In task 8, we will use a callback routine when sorting. Explain how such a callback routine works.
        
    A callback routine is a function we pass into another function so that the other function can use it while it runs. In sorting, the callback receives two elements and returns a value that tells JavaScript which element should come first.

  7. 
        What happens in the sorting routines you have created? Try to explain how the array can figure out if an element should come before or after another.
        
    The sort method compares two elements at a time by calling the callback function. If the callback returns a negative value, the first element moves before the second; if it returns a positive value, it moves after; and if it returns zero, the order between those two elements stays unchanged.
